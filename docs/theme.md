# fxPanel Theme

## Colors

### Base

| Token      | Value     | Usage             |
| ---------- | --------- | ----------------- |
| Background | `#161923` | Page background   |
| Card       | `#1c202e` | Card backgrounds  |
| Foreground | `#f1f1e4` | Primary text      |
| Muted      | `#1c202e` | Muted backgrounds |
| Muted Text | `#9ea4bd` | Secondary text    |
| Border     | `#292d3d` | Borders, dividers |

### Accent

| Token  | Value     | Usage                              |
| ------ | --------- | ---------------------------------- |
| Accent | `#f40552` | Cfx pink - highlights, focus rings |

### Semantic

| Token       | Value     | Usage           |
| ----------- | --------- | --------------- |
| Destructive | `#e33131` | Errors, bans    |
| Warning     | `#ffae00` | Warnings        |
| Success     | `#01a370` | Success, online |
| Info        | `#2b9bc5` | Info states     |

## Typography

| Token | Value                   |
| ----- | ----------------------- |
| Sans  | Inter Variable          |
| Mono  | JetBrains Mono Variable |

## Spacing

| Token   | Value  |
| ------- | ------ |
| Navbar  | 3.5rem |
| Sidebar | 16rem  |
| Radius  | 0.5rem |

## Dark/Light

Dark mode is default. Light mode inverts the base colors while keeping semantic colors consistent.
